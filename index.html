Selected: Branch.io

One-Man Shop Integration Plan

Phase 1: Admin & Config Est: 2 Hours

1. Sign up for "Launch" Plan

Requires credit card. Warning: Set a calendar reminder to check MAU volume monthly to avoid overage fees.

2. Configure Link Domain

In Dashboard, set up kickit.app.link (or similar). This prevents having to mess with DNS records on your main site.

Phase 2: Code Integration Est: 1.5 Days

1. Android: Add Dependency

Update build.gradle module file.

implementation 'io.branch.sdk.android:library:5.+'
2. iOS: Add Associated Domains

In Xcode > Signing & Capabilities. Add applinks:kickit.app.link. This is the "magic" step for Universal Links.

3. iOS: Add Keys to Info.plist

Add branch_key string and branch_universal_link_domains array.

Selected: AppsFlyer

One-Man Shop Integration Plan

Phase 1: Zero Plan Setup Est: 1 Hour

1. Create Account & Select "Zero" Plan

No credit card usually required for the free tier. Grab your Dev_Key from the dashboard settings.

2. Apple App Store Connect

You DO NOT need to change anything in the store listing yet, but you will need your Apple ID (e.g., id123456789) for the AppsFlyer dashboard.

Phase 2: Code Integration Est: 1 Day

1. iOS: Add Pod & ATT Prompt

Add pod 'AppsFlyerLib'. Crucial: You must add the code to request user permission for tracking (ATT) in AppDelegate.

AppsFlyerLib.shared().waitForATTUserAuthorization
2. Android: Permissions

Ensure INTERNET and ACCESS_NETWORK_STATE are in the Manifest. (You likely already have these).

3. The "Kitchen Test"

Install the "Debug" version on your phone. Click a test link. Verify "Non-Organic Install" appears in the dashboard within 2 minutes.

Selected: Manual (HubSpot)

One-Man Shop Integration Plan

Phase 1: Marketing Setup Est: 3 Hours

1. Create "Gateway" Landing Page

Build a simple HubSpot LP. It must have two buttons: "Download iOS" and "Download Android".

2. Button Tracking

Create a HubSpot "Call-to-Action" (CTA) button for each store. This tracks the click intent.

Phase 2: The Manual Routine Recurring: Monthly

1. Database Export

Write a SQL query or use admin panel to export all new User Emails from the App Database for the last 30 days.

2. HubSpot Import

Import CSV to HubSpot. Map emails. Update property App_Status to "Installed".
